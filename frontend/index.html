Sunrise/Sunset ICS

Generate calendar events

Pick a location and date range. Download an ICS file with sunrise/sunset events.

Tip: tap the map to set the location, or use search results above.
HowTo
  1. Search for a place or enter latitude/longitude manually.
  2. Select a date range and event duration.
  3. Choose sunrise, sunset, or both.
  4. Click “Download ICS” and import into your calendar.

Times are computed locally (no API key). Location search uses OpenStreetMap Nominatim.
